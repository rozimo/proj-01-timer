# Project Overview
Onigiri Timer is a focused countdown application and my first venture into independent, end-to-end development. While my previous experience involved collaborative academic projects, this application was built entirely from scratch to master the fundamentals of the Electron framework, full-stack integration, and custom UI implementation.

This project serves as a cornerstone for my development journey as I transition from academic exercises to building professional-grade, solo applications.

# Design & Implementation
To create a unique user experience, I integrated my background in digital illustration with functional software engineering.

## Custom UI
Every visual element and interface asset was hand-illustrated (by me) to create a cohesive, themed aesthetic.
